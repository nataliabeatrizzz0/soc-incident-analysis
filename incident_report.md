# 🚨 Incidente Reportado:

## 📅 Data
20/04/2026

## 🔎 Descrição
Foram identificadas múltiplas tentativas de login com falhas seguidas e um acesso bem-sucedido.

## ⚠️ Evidências
- 3 tentativas de login falhas;
- Login bem-sucedido após falhas;
- Acesso fora do horário padrão.

## 🧠 Análise
O comportamento indica possível ataque de força bruta ou uso de credenciais comprometidas.

## 🚨 Impacto
Possível acesso não autorizado a sistema sensível (financeiro).

## ✅ Ações recomendadas
- Bloqueio do usuário;
- Reset de senha;
- Verificação de atividade recente;
- Ativação de MFA.
